Input Table - Wiki

$(selction).inputtable({ options })
Options :
[num] rows : default 5
[num] cols : default 5
[num] minSpareRows : Maintains an empty row after the last entered cell in the table.Accounts for the auto expanding nature of the table .default 0
[num] minSpareCols : Maintains an empty column after the last entered cell in the table.Accounts for the auto expanding nature of the table .default 0
[bool] fillHandle : Inserts a handle at the bottom of each cell, for easy drag. default true
[array] colHeaders : Specify various titles for different columns in array format.If nothing is specified it assigns in alphabetical order
[bool] rowHeaders :Adds a count in the the nos of rows. defualt true
Sample Code :
$("#input-table").inputtable({ rows: 5, cols: 3, minSpareCols : 1, minSpareRows : 1, fillHandle : true });
Methods :
LoadData Array

Used to prepopulate the contents of a table with a two dimensional Array

$(selection).inputtable('loadData',Data) var data = [ ["", "Heading 1", "Heading 2", "Heading 3", "Heading 4"], ["Column 1", 10, 11, 12, 13], ["Column 2", 20, 11, 14, 13], ];

getData

Returns the content of the table in the form of a two dimensional Array

var Array = $(selection).inputtable('getData'); console.log(Array); Returns: [ ["", "Heading 1", "Heading 2", "Heading 3", "Heading 4"], ["Column 1", 10, 11, 12, 13], ["Column 2", 20, 11, 14, 13], ];

getSelectedData

Returns the content of the selected cells in the form of a multi dimensional Array

var Array = $(selection).inputtable('getSelectedData');

Clear

Clears the content of the table to a blank one

$(selection).inputtable('clear');
